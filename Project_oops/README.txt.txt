PROJECT TITLE: Bus/Train Booking System
LANGUAGE USED: C++
TOOLS USED: Code::Blocks / Dev-C++

DESCRIPTION:
This project allows users to book and cancel tickets for buses or trains.
It includes features like seat availability check, admin login, and ticket details display.

FILES INCLUDED:
1. CaseStudy.docx – Detailed project report
2. booking_system.cpp – Source code file
3. Inputs.pdf – Input samples
4. Outputs.pdf – Output results
5. README.txt – Project overview

RUNNING INSTRUCTIONS:
1. Open booking_system.cpp in any C++ IDE like Code::Blocks or Dev-C++.
2. Compile and run the program.
3. Follow the on-screen menu for booking, cancelling, or viewing tickets.
